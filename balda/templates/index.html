Welcome to Балда

This game is a popular Russian word game. There is a grid with a word in the middle. Turn by turn, each player must add a letter to the grid to form a new word. The longer the word, the higher the score. The game ends when the grid is full or when no new word is found.

This game was made by Alexandre Blin and Tatiana Kulish for the Agile Web course.
Click on the button to start a new game :

Game size: 5x5
